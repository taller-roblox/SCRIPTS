¿Qué es un Script?

Nivel 1 · Básico

Un script es como una lista de instrucciones que le das a tu computadora. Imagina que tienes un robot y le escribes: "Camina 3 pasos, gira a la derecha, salta". ¡Eso es básicamente un script! 🤖

En Roblox Studio, los scripts están escritos en un lenguaje llamado Lua. Con Lua puedes hacer que tus piezas de Roblox se muevan, cambien de color, hagan ruidos, den puntos y ¡mucho más!

💡

¡Dato curioso! Lua significa "luna" en portugués. ¡Es un lenguaje muy popular en videojuegos porque es rápido y fácil de aprender!

📝

Script Normal

Controla cosas en el servidor del juego. Todos los jugadores lo ven igual.

🖥️

LocalScript

Solo afecta al jugador en su pantalla. Perfecto para la interfaz y efectos visuales.

📦

ModuleScript

Guarda código que otros scripts pueden usar. Como una caja de herramientas compartida.

⌨️

¿Qué es una Variable?

Nivel 2 · Variables

Una variable es como una caja con un nombre donde puedes guardar información. Por ejemplo, puedes tener una caja llamada vida que guarda el número 100.

En Lua, creamos variables así: local nombre = valor

🎮

En Roblox, las variables se usan para todo: la vida del jugador, su velocidad, su nombre, el color de una parte, ¡todo!

📄 Tipos de Variables
-- Número (number) local vida = 100 local velocidad = 16 -- Texto (string) local nombre = "Jugador1" local color = "Rojo" -- Verdadero o Falso (boolean) local estaVivo = true local tieneEspada = false -- Mostrar variables print("Mi nombre: " .. nombre) print("Mi vida: " .. vida)
🔗

El símbolo .. (dos puntos) une textos y variables. Se llama concatenación. ¡Es como pegar palabras!

🔀 If, Then, Else

¡Toma decisiones en tu código!

🔀

¡Si... entonces... sino!

Nivel 3 · Condicionales

Las condiciones le permiten a tu código tomar decisiones. Imagina: "SI el jugador tiene más de 0 de vida, ENTONCES puede seguir jugando, SINO muere y respawnea". ¡Así funciona!

📄 if / then / else / end
local vida = 50 if vida > 0 then print("¡El jugador está vivo! 💪") elseif vida == 0 then print("¡Peligro! Vida en cero ⚠️") else print("¡Game Over! 💀") end -- Comparadores: -- > mayor que -- < menor que -- == igual a (¡dos signos igual!) -- ~= diferente de -- >= mayor o igual -- <= menor o igual
⚠️

¡Importante! Para comparar usas == (dos iguales). Un solo = es para guardar valores. ¡No los confundas!

🔁 Loops (Bucles)

¡Repite código sin tener que escribirlo mil veces!

🔁

¡Que se repita, que se repita!

Nivel 4 · Bucles

Un loop (bucle) repite el mismo código varias veces. ¿Imagina tener que escribir print("¡Hola!") 100 veces? Con un loop solo lo escribes una vez y le dices cuántas veces repetirlo. 🔁

📄 Loop "for" - El más usado
-- for [variable] = [inicio], [fin] do for i = 1, 5 do print("Vuelta número: " .. i) end -- Contando de 2 en 2: for i = 0, 10, 2 do print("Par: " .. i) end -- Loop "while" - mientras algo sea verdad: local vida = 3 while vida > 0 do print("Vidas restantes: " .. vida) vida = vida - 1 end print("¡Game Over!")
🚨

¡Cuidado con los loops infinitos! Si haces un while que nunca para, ¡tu juego se congela! Asegúrate de que siempre haya una forma de salir.

Funciones: Tu superpoder

Nivel 5 · Funciones

Una función es un bloque de código con nombre que puedes usar cuando quieras. Es como crear tu propio comando. En vez de escribir el mismo código 10 veces, lo pones en una función y la llamas cuando la necesites.

📄 Crear y usar funciones
-- Función simple local function saludar() print("¡Hola, jugador! 👋") end saludar() -- Llamamos la función saludar() -- La podemos llamar cuantas veces queramos -- Función con parámetros (información extra) local function darVida(jugador, cantidad) print(jugador .. " recibió " .. cantidad .. " de vida ❤️") end darVida("Herobrine", 50) darVida("Noob123", 100) -- Función que devuelve un valor (return) local function sumar(a, b) return a + b end local resultado = sumar(30, 70) print("Total: " .. resultado)

Mover y colorear partes

Nivel 6 · Roblox Real

En Roblox, todo objeto del juego (partes, modelos, personajes) se controla a través del game y el workspace. Aquí tienes ejemplos reales que puedes copiar a Roblox Studio.

📄 Cambiar color de una parte
-- Este código va en un Script dentro de una Part local parte = script.Parent -- Cambiar el color parte.BrickColor = BrickColor.new("Bright red") -- Hacerla más grande parte.Size = Vector3.new(10, 5, 10) -- Hacerla semitransparente parte.Transparency = 0.5
📄 Ladrillo que cambia de color en loop
local parte = script.Parent local colores = {"Bright red", "Bright blue", "Bright yellow", "Lime green"} while true do for i = 1, #colores do parte.BrickColor = BrickColor.new(colores[i]) task.wait(1) -- espera 1 segundo end end
📄 Puerta que se abre al tocar
local puerta = script.Parent local abierta = false puerta.Touched:Connect(function(objeto) if not abierta then abierta = true print("¡Puerta abierta! 🚪") puerta.Transparency = 0.8 puerta.CanCollide = false task.wait(3) puerta.Transparency = 0 puerta.CanCollide = true abierta = false end end)

📚 Glosario Roblox-Lua

Todas las palabras importantes en un solo lugar

local
Palabra para crear una variable. Siempre úsala para buenas prácticas.
print()
Muestra un mensaje en la consola de Output.
if / then / end
Estructura para tomar decisiones en tu código.
else / elseif
El "si no" del if. Ejecuta código cuando la condición es falsa.
for
Bucle que repite código un número específico de veces.
while
Bucle que repite código mientras algo sea verdad.
function
Un bloque de código reutilizable con un nombre.
return
Devuelve un valor desde una función.
true / false
Valores booleanos. Verdadero o Falso.
..
Concatenación: une textos o variables de texto.
script.Parent
El objeto de Roblox que contiene el script.
task.wait()
Pausa el script por X segundos.
workspace
El mundo 3D donde están todas las partes del juego.
:Connect()
Conecta un evento (como Touched) con una función.
BrickColor
El color de una parte en Roblox.
Vector3
Una posición o tamaño en 3D con valores X, Y, Z.
Transparency
Qué tan transparente es una parte. 0=sólido, 1=invisible.
CanCollide
Si los jugadores pueden atravesar la parte o no.
